Logistic Regression Model: Description: Logistic Regression is a statistical model used for binary classification tasks, such as predicting whether a user will purchase a product or not (0 = No, 1 = Yes). It estimates the probability of a binary outcome (1 or 0) based on one or more predictor variables.

In this project, we use Logistic Regression to predict whether a user will purchase a product based on their age, gender, and estimated salary.

Advantages of Logistic Regression:

Simple and easy to implement: Logistic Regression is simple to understand and implement, making it suitable for beginners. Efficient for binary classification: It works well for binary classification problems, such as predicting yes/no or true/false outcomes. Interpretable coefficients: The coefficients of Logistic Regression represent the influence of each feature on the predicted probability, making it easy to interpret. Less prone to overfitting: Logistic regression has less risk of overfitting compared to complex models like decision trees or neural networks. Disadvantages of Logistic Regression:

Assumption of linearity: It assumes a linear relationship between the input variables and the log-odds of the dependent variable. Not ideal for non-linear problems: Logistic regression may not perform well if the underlying relationships between variables are not linear. Sensitive to outliers: Outliers can significantly affect the model's performance since logistic regression doesnt handle them well. Limited to binary classification: It is primarily used for binary outcomes, so it's not suitable for multi-class classification tasks without modification.

# Import necessary libraries
import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import accuracy_score, classification_report

# Step 1: Import the dataset
df = pd.read_csv('Social_Network_Ads.csv')

# Step 2: Check column names to avoid KeyError
print("Columns in the dataset:", df.columns)

# Step 3: Exploratory Data Analysis (EDA)
# Numerical features: Age, EstimatedSalary
plt.figure(figsize=(12, 5))

# Histogram for 'Age'
plt.subplot(1, 2, 1)
sns.histplot(df['Age'], bins=20, kde=True)
plt.title('Age Distribution')

# Histogram for 'EstimatedSalary'
plt.subplot(1, 2, 2)
sns.histplot(df['EstimatedSalary'], bins=20, kde=True)
plt.title('Estimated Salary Distribution')

plt.show()

# Step 4: Splitting the dataset into training and testing sets
X = df[['Age', 'EstimatedSalary']]
y = df['Purchased']
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.25, random_state=42)

# Step 5: Feature scaling
scaler = StandardScaler()
X_train[['Age', 'EstimatedSalary']] = scaler.fit_transform(X_train[['Age', 'EstimatedSalary']])
X_test[['Age', 'EstimatedSalary']] = scaler.transform(X_test[['Age', 'EstimatedSalary']])

# Step 6: Model training
model = LogisticRegression()
model.fit(X_train, y_train)

# Step 7: Prediction
y_pred = model.predict(X_test)

# Step 8: Performance metrics - C04
# Accuracy Score
accuracy = accuracy_score(y_test, y_pred)
print(f"Accuracy: {accuracy:.2f}")

# Classification Report
report = classification_report(y_test, y_pred)
print("Classification Report:")
print(report)

# Optional: Visualize feature importance (coefficients)
coefficients = pd.DataFrame(model.coef_, columns=X.columns)
plt.figure(figsize=(10, 6))
sns.barplot(x=coefficients.columns, y=model.coef_.flatten())
plt.title('Feature Coefficients')
plt.xticks(rotation=45)
plt.show()
